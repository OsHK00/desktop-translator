# desktop-translator

desktop-translator is a small translation application repository.

## Overview

desktop-translator is intended to translate text between languages.

## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/OsHK00/desktop-translator.git
    ```
2. Change into the project directory:
    ```bash
    cd desktop-translator
    ```
3. Install dependencies:

    ```bash
    pip install -r requirements.txt
    ```
